# Visión del Proyecto Final: El Sistema de Inteligencia de Mercado

El objetivo de este proyecto es integrar todos los conocimientos adquiridos en el curso para construir un sistema real, profesional y escalable que combine **Ingeniería de Datos**, **Backend con Python** e **IA Generativa**.

## 1. El Concepto
Vamos a construir un **Analista de Mercado Automatizado**. El sistema debe ser capaz de:
1.  Ingerir noticias y datos financieros de diversas fuentes.
2.  Almacenarlos de forma eficiente y segura.
3.  Procesarlos para extraer sentimientos y entidades clave usando IA.
4.  Exponer los resultados a través de una API profesional.
5.  Permitir a los usuarios hacer preguntas en lenguaje natural sobre los datos (RAG).

## 2. Por qué este proyecto
Este proyecto no es un "hola mundo". Es un sistema que demuestra:
*   Capacidad para manejar **Docker** y despliegues reproducibles.
*   Dominio de **FastAPI** y seguridad (JWT).
*   Diseño de **Pipelines ETL** robustos.
*   Implementación de arquitecturas de **IA modernas (RAG + Agentes)**.
*   Prácticas de **Observabilidad** y **Performance**.

## 3. Resultado Esperado
Al finalizar, tendrás un repositorio profesional listo para tu portfolio, con documentación completa, tests automatizados y una arquitectura que sigue los estándares de la industria (Software 3.0).
