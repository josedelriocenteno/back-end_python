# Checklist: Dominas Fundamentos de Python – Backend Profesional

## 1. Sintaxis y estructura

- [ ] Entiendo indentación y legibilidad en Python.
- [ ] Sé diferenciar scripts de módulos.
- [ ] Uso comentarios y docstrings correctamente.
- [ ] Aplico un estilo de código consistente y profesional.

---

## 2. Tipos de datos y conversiones

- [ ] Conozco tipos básicos: int, float, bool, None, str.
- [ ] Sé hacer conversiones y castings seguros.
- [ ] Evito errores comunes de tipos en operaciones.

---

## 3. Estructuras de datos

- [ ] Sé usar listas, tuplas, sets y diccionarios adecuadamente.
- [ ] Comprendo complejidad básica de cada estructura.
- [ ] Aplico manipulación eficiente de datos (append, pop, lookup).

---

## 4. Control de flujo

- [ ] Sé usar if, elif, else correctamente.
- [ ] Sé usar for, while, break, continue y pass.
- [ ] Evito errores lógicos comunes en bucles y condicionales.

---

## 5. Funciones

- [ ] Creo funciones claras, pequeñas y enfocadas.
- [ ] Uso argumentos posicionales, nominales y por defecto correctamente.
- [ ] Comprendo funciones puras y efectos secundarios.
- [ ] Aplico buenas prácticas en funciones (nombres, docstrings, modularidad).

---

## 6. Manejo de errores

- [ ] Sé usar try, except, else y finally correctamente.
- [ ] Conozco excepciones comunes y cómo crearlas.
- [ ] Diferencio errores controlados de crasheos inesperados.

---

## 7. Módulos y paquetes

- [ ] Comprendo import, from y la estructura de paquetes.
- [ ] Sé usar __init__.py y modularización profesional.
- [ ] Mantengo el código organizado en paquetes coherentes.

---

## 8. Input / Output

- [ ] Sé leer y escribir archivos de forma eficiente.
- [ ] Comprendo rutas, paths y manejo de errores en I/O.
- [ ] Evito errores comunes de lectura/escritura y encoding.

---

## 9. Strings avanzados

- [ ] Uso slicing, f-strings y métodos clave de strings.
- [ ] Comprendo diferencias entre strings y bytes.
- [ ] Aplico strings de manera profesional en backend y datos.

---

## 10. Iterables y comprehensions

- [ ] Comprendo iterables vs iteradores.
- [ ] Uso list, dict y set comprehensions correctamente.
- [ ] Mantengo legibilidad evitando “magia” innecesaria.
- [ ] Evito errores comunes en comprehensions complejas.

---

## 11. Funciones built-in clave

- [ ] Uso sum, min, max, any, all, map, filter, zip correctamente.
- [ ] Sé cuándo aplicarlas y cuándo no.
- [ ] Optimizo operaciones con estas funciones en pipelines de datos.

---

## 12. Complejidad y rendimiento básico

- [ ] Comprendo Big O y notación de complejidad.
- [ ] Sé medir y comparar coste de operaciones.
- [ ] Tomo decisiones eficientes desde el inicio (evitando O(n^2) innecesario).

---

## 13. Buenas prácticas generales

- [ ] Código legible y mantenible.
- [ ] Evito código spaghetti y variables globales.
- [ ] Aplico mentalidad profesional: planificación, tests, modularidad.
- [ ] Uso linters, formatters y documentación desde el día 1.

---

## ✅ Regla de oro

> Antes de continuar con temas avanzados, asegúrate de que **cada item de este checklist esté dominado**.  
> Fundamentos sólidos = base para backend escalable, pipelines eficientes y código profesional.
